# 🚀 Project Management Tool

A full-stack MERN Project Management Tool built using React, Node.js, Express, and MongoDB.

## Features

### Authentication

* User Registration
* User Login
* JWT Authentication
* Logout

### Project Management

* Create Project
* View Projects
* Edit Project
* Delete Project

### Task Management

* Create Task
* View Tasks
* Update Task Status
* Todo / In Progress / Completed

### Dashboard

* Total Projects Count
* Total Tasks Count
* Completed Tasks Count
* Analytics Dashboard

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
